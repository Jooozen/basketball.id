# Basketball Strategy Board

Interactive basketball strategy board web application for coaches and players.

## Features

- **Court Views**: Switch between full court and half court
- **Draggable Players**: 5 offense (blue), 5 defense (red), and a ball piece
- **Drawing Tools**: Freehand, arrows, dashed lines, eraser, text labels
- **Preset Formations**: 1-3-1, 2-3 Zone, 3-2 Zone, Man-to-Man, Horns, Box
- **Customization**: 6 pen colors, adjustable line width
- **Undo/Redo**: Ctrl+Z / Ctrl+Y keyboard shortcuts
- **Export**: Save the board as a PNG image
- **Touch Support**: Works on mobile and tablet devices

## Usage

Open `index.html` in any modern web browser. No build tools or server required.

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl+Z   | Undo   |
| Ctrl+Y   | Redo   |
